🔐 암호화 메모장

🛡️ 보안 기능

• AES 암호화: 군사급 암호화 알고리즘 사용

• 로컬 저장: 브라우저에 암호화된 상태로 안전하게 저장

• 비밀번호 보호: 올바른 비밀번호로만 복호화 가능

🔒 개인정보 보호: 모든 데이터는 사용자의 브라우저에만 저장되며, 서버로 전송되지 않아 완전히 안전합니다!

📖 사용법 가이드

1단계: 비밀번호 입력란에 원하는 비밀번호를 설정하세요

2단계: 텍스트 영역에 보호하고 싶은 메모를 작성하세요

3단계: '💾 저장' 버튼을 클릭하여 암호화된 메모를 저장하세요

4단계: 메모를 확인하려면 같은 비밀번호를 입력하고 '📂 불러오기'를 클릭하세요

✨ 특징

• 실시간 암호화/복호화

• 비밀번호 없이는 절대 복호화 불가

• 깔끔하고 직관적인 인터페이스

• 다양한 암호화 강도 지원

💡 팁: Enter 키를 누르면 자동으로 저장 또는 불러오기가 실행됩니다!

🔒 보안 정보

🏛️ AES 암호화란?

Advanced Encryption Standard의 약자로, 미국 정부와 전 세계에서 사용하는 대칭키 암호화 표준입니다.

⚡ 성능 특징

• 빠른 속도: 실시간 암호화/복호화

• 강력한 보안: 256비트 키 사용

• 효율성: 최소한의 리소스 사용

• 안정성: 수많은 검증을 거친 알고리즘

💾 데이터 저장

모든 암호화된 데이터는 브라우저의 로컬 스토리지에 저장됩니다. 외부 서버나 클라우드로 전송되지 않아 완전한 프라이버시를 보장합니다.

⚠️ 주의사항: 비밀번호를 잊어버리면 메모를 복구할 수 없습니다. 안전한 곳에 보관하세요!
※ 정확히 RESET 입력 시에만 삭제됩니다.
모든 암호화된 메모가 완전히 삭제됩니다.
